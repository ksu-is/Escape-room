# Escape-room
Escape the Dorm is a text‑based adventure game where the player must find their keys, backpack, and shoes before time runs out. Navigate through different areas, make choices, and escape the dorm before you're late to class.

![Dorm Room](images/dormroom.png)


## How to Run
1. Install Python 3
2. Download or clone this repository
3. Run the game with: Python main.py

## Features
- Multiple locations (Dorm Room, Hallway, Outside)
- Item searching and simple inventory system
- Countdown timer
- Multiple endings based on player choices

## How It Works
The game uses functions, loops, and conditional logic to let the player navigate rooms, search for items, and manage time. Each decision affects the outcome of the game.

## Purpose
This project was created to practice Python fundamentals while building a fun and interactive software experience.
